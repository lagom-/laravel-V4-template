laravel-start-up
================

# Installation
## Step 1:
`$ curl -sS https://getcomposer.org/installer | php-cgi-5.4`

## Step 2:
`$ git clone git://github.com/lagom-/laravel-start-up.git`

## Step 3: 
`$ php-cli-5.4 composer.phar install`

## Step 4: 
`$ php-cli-5.4 composer.phar update`

Step 5:
`$ php-cli-5.4 artisan:key generate`
